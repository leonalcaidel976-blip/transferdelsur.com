# Transfer del Sur Costa del Sol - PRD

## Problema Original
El usuario quiere mejorar el posicionamiento SEO de www.transferdelsur.com para aparecer en los primeros resultados de Google cuando la gente busque "transfer".

## Decisiones del Usuario
- Crear versión nueva con SEO optimizado
- Mantener el diseño actual
- Palabras clave principales: transfer aeropuerto Málaga, VTC Costa del Sol, transfer Marbella, luxury vehicles Malaga, tours

## Arquitectura
- **Frontend**: React.js con Tailwind CSS
- **Fuentes**: Cormorant Garamond (headings), Montserrat (body)
- **Tema**: Luxury dark con acentos dorados (#c9a962)
